quiZone Software Package overview

Three main applications to perform different tasks helps to create interactive environment and bring new experience to audience.

quiZone BUILDER

quiZone BUILDER IS THE QUIZONE CREATION SOFTWARE

Builder is perfect for creating unique quiz-content. Through simple actions images, text or audio can be added. No coding skills are required to create unique content.

quiZone ENGINE

quiZone ENGINE IS THE OUTPUT SOFTWARE

ENGINE is a powerful realtime software build for maximum scalability, performance and reliability. It installed on a Host's computer and handles the actual output and display of quiZone content.

quiZone STUDIO

quiZone STUDIO IS A PLAYOUT CONTROL APPLICATION

STUDIO is web-based application for control and editing game in real time. It is easy to add quizones to playlist, preview content and check players activity.
